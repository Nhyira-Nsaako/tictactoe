# Tic-Tac-Toe Game

Welcome to the Tic-Tac-Toe game! This is a simple implementation of the classic game where two players take turns marking spaces in a 3x3 grid, trying to get three of their symbols in a row, column, or diagonal.

## How to Play

To play the game, follow these steps:

1. Clone the repository to your local machine.
2. Navigate to the directory where the game is located.
3. Run the `tictactoe.py` file with Python.
4. Follow the on-screen instructions to play the game.

## Features

- Two-player mode: Play against a friend and compete to win.
- Single-player mode: Play against the computer and test your skills.
- Simple user interface: Easy-to-use interface for smooth gameplay.
- Winning detection: The game detects and announces the winner when a player achieves three in a row.
- Tie detection: The game detects and announces a tie when the board is full and no player has won.

## Files

- `tictactoe.py`: Contains the main code for the Tic-Tac-Toe game.
- `README.md`: This file, providing information about the game and how to play it.

## Usage

To run the game, make sure you have Python installed on your machine. Then, execute the following command in your terminal or command prompt:tictactoe.py

Follow the on-screen instructions to choose the game mode and start playing.

Enjoy the game!
